The Climbers Journal's core value proposition is the **unified training view** — seeing climbing sessions AND endurance activities (run, bike, hike, walk) in one place. No existing app does this.

**Why:** The user trains as a whole athlete (climber + runner + cyclist) and needs to see all training together. Currently split across Garmin Connect, intervals.icu, Mountain Project, The Crag, Vertical Life.

**How to apply:**
- Every view (dashboard, calendar, log) should show BOTH climbing and endurance data side by side
- Effort metrics differ by domain: endurance = time/distance/HR, climbing = route count + difficulty. Don't force one metric system — show each domain in its native units.
- Future "coach" feature should reason about the whole athlete, not just climbing or just endurance
- The copilot already has access to both data sources — this is the differentiator
