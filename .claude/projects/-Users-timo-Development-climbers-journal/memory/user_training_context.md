The user is a climber and endurance athlete who currently uses:
- **Garmin Connect** — wearable syncs automatically, but limited UI
- **intervals.icu** — synced from Garmin, better analysis, training plans, has API (already integrated)
- **Mountain Project** — outdoor climbing ticks (no free API)
- **The Crag** — outdoor climbing ticks (no free API)
- **Vertical Life** — climbing ticks (no free API)
- **Strava** — social sharing (low priority)

Has a harmonized CSV file of climbing data from a blog article. No CSV currently in the repo — lives externally.

Climbing level: working outdoor 7b+ projects, hasn't sent one yet. Important distinction between "goes" (attempts) and "sends" for grade pyramid display — showing attempted grades as sends would give a wrong impression.

Prefers grade pyramid to filter: sends only (onsight/flash/redpoint/repeat), indoor vs outdoor toggle.
